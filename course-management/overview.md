# Course Management Overview

## At a glance

- Open the Hub Control Panel.
- Access Course Administration.
- Request a new course or course changes.
- Manage roster updates.

Course Management tools allow you to create and update courses, manage your course roster, and support students throughout the semester.

To access these tools from JupyterLab:

1. Select **File** from the menu bar.
2. Choose **Hub Control Panel**.

![JupyterLab File menu with Hub Control Panel option]({{ site.baseurl }}/assets/images/screenshots/jupyterlab-hub-control-panel.png){: .figure }

<div class="figure-caption">
<strong>Figure 4.</strong> Access the Hub Control Panel from the File menu in JupyterLab.
</div>

{: .expect }
> Selecting **Hub Control Panel** returns you to the JupyterHub home page, where you will find the course management tools available to you.

![Hub Control Panel course management menu]({{ site.baseurl }}/assets/images/screenshots/hub-control-panel-course-management.png){: .figure }

<div class="figure-caption">
<strong>Figure 5.</strong> The Hub Control Panel provides access to the course management tools available to you.
</div>

From the Hub Control Panel, you can access:

- **Course Administration** — View student activity and support students.
- **New Course Request** — Request a new course or changes to an existing course.
- **Roster Updates** — Add or remove students from your course.

{: .good-to-know }
> Most course management activities begin from the Hub Control Panel. When you need to manage your course, start with **File → Hub Control Panel**.
