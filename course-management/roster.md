# Managing Your Course Roster

The **Roster Updates** page allows you to add or remove students from your course throughout the semester.

Use this page whenever a student:

- Adds your course after it has been created.
- Drops your course.
- Needs access before the official roster has been updated.
- Needs to be removed from the course.

## Updating your roster

To update your roster:

1. Select the course you would like to update.
2. Enter one **NetID per line** in either the **Add students** or **Remove students** section.
3. Add comments if needed.
4. Click **Submit Roster Update**.

You can add and remove students in the same request.

{: .good-to-know }
> Roster updates use Franklin & Marshall NetIDs, with one NetID per line. If a username is not recognized, it will be ignored, allowing the remaining valid roster updates to continue processing.

{: .expect }
> After you submit a roster update, Research Computing Services will review your request. If we have questions or need additional information, we will contact you by email. Once the roster update has been completed, we will send an email confirming the requested changes.

## Preparing a course for a new semester

At the bottom of the page is a **Remove All Students** option.

This feature removes all students from the course roster while leaving instructors, course settings, software, and course materials unchanged. It is especially useful when reusing a course for a future semester.

{: .good-to-know }
> Removing a student from a course prevents them from accessing the course, but it does **not** delete any of their notebooks or files. If they are added back to the course later, their work will still be available.

{: .good-to-know }
> Using **Remove All Students** clears the roster only. It does not delete the course, remove instructors, or change the course software or configuration.
