# Requesting a New Course or Course Changes

Use the **New Course Request** page whenever you need to create a new course or request changes to an existing one.

Use this page to:

- Create a new course for an upcoming semester.
- Create a new course based on an existing approved course.
- Request additional software packages or programming libraries.
- Update or remove software in an existing course.
- Request other changes to your course environment.

## Completing your request

The request form collects the information Research Computing Services needs to prepare your course.

Depending on the type of request, you may be asked to provide:

- Your course code.
- An existing approved course to use as a starting point, if applicable.
- Information about software, packages, or other changes you would like to request.
- Faculty members who should have instructor access.
- Additional comments that may help us understand your request.

You **do not need to complete every field for every request**. Provide the information that is relevant to the changes you would like to make.

{: .good-to-know }
> You do **not** need to provide a list of students when creating a course or requesting changes to an existing course. Student access is managed separately using the **Roster Updates** tool after the course has been approved or updated.

{: .good-to-know }
> When requesting software, include as much information as possible, such as package names, programming languages, JupyterLab extensions, system tools, data files, GitHub repositories, or installation instructions.

{: .expect }
> After you submit your request, Research Computing Services will review it. If we have questions or need additional information, we will contact you by email. Once your request has been completed, we will send another email letting you know your new course or requested changes are ready to use.

{: .help }
> Not sure whether your request requires a new course or simply a change to an existing course? Submit the request anyway. If we have questions or think there is a better approach, we will contact you before making changes.
