# Before You Teach Checklist

Use this checklist before the first day your students use F&M JupyterHub.

- [ ] My F&M JupyterHub account is active.
- [ ] My course has been created or updated.
- [ ] Required software is available.
- [ ] I have opened and tested my notebooks in F&M JupyterHub.
- [ ] Students have been added to the course roster.
- [ ] Canvas links have been tested.
- [ ] If using GitHub, my nbgitpuller links work as expected.
- [ ] I know how to reach Research Computing Services if help is needed.

{: .good-to-know }
> Testing the student experience before class can help prevent confusion during the first assignment or activity.
