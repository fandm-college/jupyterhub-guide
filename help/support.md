# Need Help?

Research Computing Services can assist with:

- JupyterHub
- Course creation
- Course software updates
- Technical issues, such as a student being unable to log in

To request assistance:

1. Visit [https://request.fandm.edu](https://request.fandm.edu).
2. Search for **Research Computing**.
3. Select any Research Computing service.
4. Submit your request with as much detail as possible.

Your request will be routed to Research Computing Services.

{: .help }
> If you are unsure which option to choose, select any Research Computing service. The request will be routed to Research Computing Services.
